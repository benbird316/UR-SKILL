# Examples

> **Purpose**: Production-grade examples bundled with UR-SKILL itself, all designed and validated using the UR-SKILL methodology. Ready for direct reference or reuse.
> **Core Principle**: Examples are fully runnable SKILLs/Agents, not structural diagrams.

---

| Example | Location | Type | Features | Best Reference For |

## Example 4: UR-SKILL Itself (Complete Production-Grade SKILL Package)

**Location**: The currently installed language directory (i.e., the directory containing `SKILL.md`)

**Core Domain**: SKILL Generation Engineering

**Reference Value**:
- Standardized directory structure for a full SKILL package
- Concrete implementation of capability matrix 4-layer design (Core Domain + 6 Radiating Domains x 4 layers)
- Full 4-master-node, 13-sub-step workflow (Analyze -> Execute -> Reflect -> Deliver)
- 6-dimension checklist writing for gate nodes (Steps 10-12: Validate -> Verify -> Loop Decision)
- Organization and cross-referencing of 15+ design guides
- Example filling of 9 template files
- Capability matrix and workflow design for 3 sub-Agents
- Blind Spot Three-Layer Mechanism, Loop Principle, Risk Boundary Declaration
- Self-contained references (all `./` references point to files within the SKILL package)

| Your Scenario | Recommended Reference |

| Create a code / script skill | Script Engineer Agent |
| Create a complete production-grade SKILL package | UR-SKILL's own SKILL.md + directory structure |
| Learn how design guides are organized | UR-SKILL's design-guides/ directory |
| Learn how workflow gate controls are written | UR-SKILL's Steps 10-12 (Validate -> Verify -> Loop Decision) |
