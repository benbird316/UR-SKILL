# 反模式文件模板

> 用途：定义生成 SKILL 的 anti-patterns.md 标准结构
> 核心原则：反模式必须解释"好心→办坏事→为什么→怎么做对"的完整教学链条
> 设计方法详见 [design-guides/anti-patterns-design-guide.md](../design-guides/anti-patterns-design-guide.md)

---

## 1. 通用反模式引用

通用反模式（占位符残留、规范越位、示例污染、架构混淆、检查缺失、盲区逃避、风险边界滥用、切面填充、盲区甩锅）由 UR-SKILL 统一处理。本文件只列出该 SKILL 领域特有的反模式。

> 通用反模式在 UR-SKILL 的 References/anti-patterns.md 中统一定义；生成 SKILL 时禁止重复罗列。

## 3. 完整性检查清单

- [ ] 只包含领域特有反模式，不含通用反模式
- [ ] 每个反模式包含表现、为什么出现、危害、检测方法、规避策略
- [ ] 检测方法具体可执行
- [ ] 规避策略指向正确做法
- [ ] 文件 < 200 行
