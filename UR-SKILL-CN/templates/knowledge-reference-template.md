# 知识型参考文件模板

> 用途：定义生成 SKILL 的知识型参考文件（knowledge-reference）标准结构
> 核心原则：知识型文件是"知道什么"，不是"怎么做"或"什么不能做"
> 设计方法详见 [design-guides/knowledge-reference-design-guide.md](../design-guides/knowledge-reference-design-guide.md)
> 子类型：K1 领域知识 / K2 API&CLI 参考 / K3 配置&策略 / K4 设计模式
