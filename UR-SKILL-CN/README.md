# UR-SKILL (Chinese Edition)

**Meta-SKILL &mdash; Design Agent SKILLs with Engineering Precision.**

UR-SKILL turns vague ideas like &ldquo;I need a code review SKILL&rdquo; into production-grade AI Agent SKILL packages. Built on a 7-step engineering workflow, 6-facet review system, and 3 cognitive state model.

This is the **Chinese-language edition** of UR-SKILL.

## Quick Install

```bash
python install.py --lang zh-cn [--target <platform>] [--name <name>]
```

Supported platforms: Claude Code, Cursor, Trae, VS Code Copilot, Gemini CLI, Windsurf, Cline, and 30+ more.

## Validation

```bash
python UR-SKILL-CN/Scripts/validate_skill.py --skill-dir <path>
```
